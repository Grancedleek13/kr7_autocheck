Контрольная работа (7 класс) — автопроверка (как в "первом экземпляре")

Файлы:
- kr_app.py — приложение (окно с заданиями)
- questions.json — вопросы/ответы (можно редактировать учителю)

Запуск (Windows / VS Code):
1) Установите Python 3.10+.
2) Откройте папку в VS Code.
3) Запустите:
   python kr_app.py

Проверка:
- Ученик вводит ответы в поля/выбирает варианты.
- Кнопка "Проверить" показывает ✅/❌ и (как в первом экземпляре) подсказку с правильным решением для неверных ответов.
